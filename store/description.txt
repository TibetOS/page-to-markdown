Turn any web page into clean, LLM-ready Markdown — download it, copy it, or send it straight to your tools. Everything runs locally.

Page to Markdown uses Mozilla's Readability engine to strip away ads, navigation, sidebars, and other noise, then converts the clean article to well-formatted Markdown with proper headings, links, images, code blocks — and even LaTeX math.

WHAT YOU CAN DO
• Extract .md — download the page as a Markdown file with rich YAML front matter
• Copy Markdown — full Markdown straight to your clipboard, with a token estimate
• Copy for AI — token-lean Markdown (no front matter, no images) ready to paste into ChatGPT, Claude, or any LLM chat
• Clip all tabs — every open https tab into one combined Markdown file (permissions asked only on first use)
• RAG chunks — export the page as heading-structured JSONL, ready for embedding pipelines
• Tables → CSV — download every table on the page as a clean CSV file
• Send to Obsidian — create a note in your vault in one click (optional default vault and folder, or append to today's daily note)
• Send to webhook — POST extractions as JSON to your own endpoint (n8n, Zapier, Make, self-hosted)
• Preview & edit — review and tweak the Markdown before saving, with live word/token counts
• Copy a selection — highlight text, right-click, get just that part as Markdown
• Keyboard shortcut & right-click menu — skip the popup entirely (Ctrl/Cmd+Shift+M)
• English & Hebrew UI — full RTL layout included

RICH, CONFIGURABLE FRONT MATTER
Each extraction can include title, author, source URL, site, publish date, language, excerpt, and timestamp — every field togglable in settings, and always safely escaped so titles with quotes or colons never break your YAML. Perfect for Obsidian, static site generators, and RAG pipelines.

BUILT FOR AI WORKFLOWS
Markdown is the native language of LLM context. "Copy for AI" gives you token-lean output with a source line and a token estimate, so you know the context cost before you paste.

ON-DEVICE AI (OPT-IN)
With Chrome's built-in Gemini Nano, Page to Markdown can add a TL;DR summary and 3–6 topic tags to your front matter, and clean up hard pages — a 🧹 button in the preview flags leftover menus, cookie banners, and footer junk, and removes them without ever rewriting your content. All of it computed entirely on your device: off by default, feature-detected, and the model is only ever downloaded when you press the download button in settings. No cloud AI, ever.

TECHNICAL CONTENT DONE RIGHT
• Fenced code blocks with language tags
• KaTeX/MathJax formulas recovered as $…$ / $$…$$ LaTeX
• Image deduplication for lazy-loading pages
• Hebrew & RTL support — full Unicode filenames and content

PRIVACY FIRST
No accounts, no analytics, no cloud APIs, no data collection. The extension touches a page only when you invoke it (activeTab). If you configure a webhook, the browser asks you to approve access to that one site — nothing is granted by default. The source code is fully open on GitHub.
