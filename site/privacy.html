Privacy Policy

Last updated: March 27, 2026

Overview

Page to Markdown is a Chrome extension that extracts web page content and converts it to Markdown format. It is designed with privacy as a core principle — no data is collected, stored, or transmitted.

Data Collection

Page to Markdown does not collect any data. Specifically:

  • No personal information is collected
  • No browsing history is recorded
  • No analytics or tracking scripts are used
  • No cookies are set
  • No data is sent to any server

How It Works

When you click the extension icon and choose to extract, the extension reads the content of your current tab using the activeTab permission. This content is processed entirely within your browser using bundled libraries (Readability.js and Turndown.js). The resulting Markdown file is downloaded directly to your device. No intermediate servers are involved.

Permissions

The extension requests only two permissions:

  • activeTab — allows reading the current page's content, only when you click the extension icon
  • scripting — allows injecting the content extraction script into the active tab

Both permissions are scoped to the active tab and are only invoked by explicit user action.

Third-Party Services

Page to Markdown does not use any third-party services, APIs, or cloud infrastructure. All processing is performed locally in your browser.

Changes to This Policy

If this policy changes, the updated version will be posted on this page with a revised date.
