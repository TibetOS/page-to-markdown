Chrome Extension · v1.0

Page to Markdown

One-click extraction of any web page's content as a clean Markdown file. Powered by Mozilla Readability. Everything runs locally — no data leaves your browser.

Install from Chrome Web Store

Smart Extraction

Mozilla Readability strips ads, nav, and noise — the same engine behind Firefox Reader View.

Clean Markdown

ATX headings, fenced code blocks, proper links. YAML front matter with title, author, URL, and timestamp.

100% Local

Zero cloud dependencies. No analytics. No data collection. Everything runs in your browser.

How it works

1 Navigate to any page
2 Click the extension icon
3 Hit "Extract .md" — done
